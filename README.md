# Image-processing-app
An application used to process and restore damaged images.

Instructions:
when uploading an image for the first time, it must be converted to gray scale, it's not necessary to convert it again after pressing reset.

Features:

Filters:

Median
Max
Min
Average
Equalization
Laplacian
Gaussian
sharp
Unsharp
Midpoint
Low_pass
High_pass
Gaussian
Geometric_mean


Noises:

salt_and_pepper
Gaussian
Speckle
Uniform
Poisson


Segmentation:

Global_thresholding
Adaptive_thresholding
Erosion
Dilation
Opening
Closing
